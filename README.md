## 🚀 Overview

The **Data Analytics Automation System** is an enterprise-style web application that automates repetitive data analytics tasks using a structured ETL pipeline.

Users can upload CSV datasets, automatically validate and clean the data, perform transformations, generate business insights, visualize key metrics through interactive dashboards, and export professional reports in Excel and PDF formats.

The project demonstrates real-world concepts used in modern analytics platforms such as automated data pipelines, reporting systems, dashboard development, and business intelligence workflows.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/data-analytics-automation-system.git
```

Navigate into the project

```bash
cd data-analytics-automation-system
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://localhost:5000
```
